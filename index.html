Synagogue Management System This application is built using Material Dashboard Angular, customized for synagogue management. It helps manage donations, user roles, contact messages, and more within a synagogue.

This admin panel uses Google's Material Design principles for a fresh, modern UI experience. The application is built with Angular and follows a modular approach, making it scalable and maintainable for future updates.
